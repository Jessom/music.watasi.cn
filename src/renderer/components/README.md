# 目录结构
```bash
┌─ bar            # 基本都是菜单组件
├   ├─ AsideBar.vue     # 左侧菜单列表
├   ├─ FooterBar.vue    # 底部播放区域
├   ├─ ItemBar.vue      # 标题，就是左边文字加向右的图标
├   ├─ MenuBar.vue      # 顶部菜单分类组件
├   ├─ ToolBar.vue      # 工具栏目，就是【歌单】那里的菜单组件
├   └─ Topbar.vue       # 重写mac windows菜单区域
├─ cell           # 列表自己
├   └─ BlendCell.vue    # 图文列表
└─ Banner.vue     # 轮播组件
```
